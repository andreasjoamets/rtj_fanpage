Artistist Diskograafia Intervjuud

Run the Jewels

Run the Jewels on hip hop duo, kuhu kuuluvad räppar Michael "Killer Mike" Render ja produtsent/räppar Jaime "El-P" Meline. 2013. aastal loodud ansambel on avaldanud kolm albumit ja ühe remix-albumi. Ansambli liikmed tutvusid 2012. aastal, kui El-P tootis Killer Mike'i hitiks tõusnud albumi. 2013. aastal avaldas Run the Jewels oma esimese albumi, mida peeti selle aasta parimaks hip-hop albumiks. 2014. aastal avaldasid nad oma teise plaadi, RTJ 2. Hiljem korraldas El-P kampaania "Kickstarter", kus ansambel remix-albumi jaoks raha kogus. Albumi tootlus annetati heategevuseks 2016. aasta jõulude ajal avaldasid nad RTJ 3 albumi ja jõulusindi, A Christmas Fucking Miracle.
